# UNMSM Research Methods - Deivhy Torres

Doctoral course project for *Research Methods and Scientific Integrity in AI and Advanced Technologies* at UNMSM.

**Author:** Deivhy Torres  
**Tentative topic:** Child anemia in Peru, with a focus on children aged 6 to 35 months and national trends from 2019 to 2024.

## Repository Structure

- `01_paradigm/` - Paradigm justification statement
- `02_method/` - Method-fit matrix
- `03_protocol/` - Research protocol draft
- `04_literature/` - Systematic review, PRISMA flow, and gap analysis

This repository will be expanded session by session as the course progresses.
